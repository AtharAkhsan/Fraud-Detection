Project Title: Corporate Financial Risk Detection System

Overview: An end-to-end Machine Learning solution designed to detect money laundering and fraudulent transactions in mobile money transfers. Built to simulate Forensic Technology workflows used in Big 4 audit firms.

Key Features:

Forensic Analytics: Flags high-risk patterns (e.g., immediate account emptying after large transfers).

AI Engine: Uses Backpropagation Neural Network with 99% Recall on test data.

Interactive Dashboard: Streamlit interface for real-time audit simulation.

Tech Stack: Python, MySQL, Scikit-Learn (MLPClassifier), Streamlit, Plotly.

Results: Identified $631M+ in potential fraudulent transactions from a sample dataset of 5,000 records.
